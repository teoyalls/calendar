基本兼容IE8的日历时间插件

插件说明

插件版本：v1.0（测试版）

简要说明：本插件是提供了四种常见状态下（只选择年月日、只选择年月、选择年月日+时分秒、只选择时分秒四种情况）的日历时间选择插件，该插件开发了一些配置参数和相关的回调函数。

接口说明

API 说明 类型 默认值
skin 自定义class样式名称，会直接add到调用的元素上 string 无
type 日历类型（1~4四种日历类型）：1（年月日类型日历）；2（年月类型日历）；3（年月日+时分秒日历类型）；4（时分秒时间类型） int 1
calenWidth 日历对象的自定义宽度，默认以调用对象的宽度为宽度，但是受最小宽度（205px）的限制 int 205
calenHeight 日历对象的自定义高度，默认是随日历内容变化而变化 int 无
defaultTime 初始化默认时间（可以是单纯的年月日格式，也可以是年月日+时分秒格式）例如：2018/01/01，或者2018/01/01 09:12:00 string 系统当前日期时间
beforeCreatFn 回调函数。日历未创建挂载到页面时触发，如果在改回调函数内添加“return false;”，则会阻止日历继续执行创建。 object 无返回参数
afterCreatFn 回调函数。日历对象初始化创建，并挂载于页面时触发，提供一个参数（日历整体的的对象元素） object 返回一个参数
changeTimeFn 回调函数。在选择了具体的日历日期时间等，点击确定或今日时间快捷选择，两个按钮所触发的回调函数，并返回一个参数（当前用户选择的具体时间），具体开发者可进行将该事件填写到需要填写的地方等操作。 object 返回一个参数
